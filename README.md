# 🛠️ Taskbar Hero Craft Simulator & Database

タスクバーヒーロー（Taskbar Hero）のクラフトガチャ収益シミュレーター ＆ 統合データベースのファンサイト用プロジェクトです。
ゲーム内マーケットの価格・取引履歴データと連携し、レシピごとの期待値（EV/ROI）、モンテカルロシミュレーション、ステージドロップ情報の提供など、高度な分析をリアルタイムで行うことができます。

---

## 📂 構成と主要ファイル

- **`index.html`**: シミュレーターのメイン画面（HTML5 / Vanilla CSS / Vanilla JS）。
- **`css/design-system.css`**: カスタムスタイルおよびお気に入り・タブナビゲーションの調整用スタイル。
- **`data/data.js`**: ゲーム内の固定マスタデータ（アイテム、レシピ、チェストドロップ構成）。
- **`data/market_db.js`**: 自動収集された最新のSteamコミュニティマーケット価格・取引履歴データベース。
- **`data/market_init.js`**: 接続エラー時のフォールバック用初期マーケット価格定義。
- **`scripts/`**: データ収集および同期バッチスクリプト群。
  - `initial_fetch.js`: 全アイテムの初期価格・全取引履歴をSteam APIから収集しデータベースを初期構築するスクリプト。
  - `daily_sync.js`: 日次で価格と直近履歴を差分アップデートする増分同期スクリプト。

## 🚀 公開・デプロイ方法

本プロジェクトは **完全な静的Webアプリケーション** です。特別なWebサーバー（Node.js / Python等）を本番環境で動かす必要はありません。

### 1. 静的ホスティングサービスでデプロイする
以下のファイルをホスティングサービス（GitHub Pages、Vercel、Netlify、Cloudflare Pagesなど）の公開フォルダにそのままアップロードしてください。

**アップロードが必要な最小構成:**
```text
├── index.html
├── css/
│   └── design-system.css
└── data/
    ├── data.js
    ├── market_db.js
    ├── market_init.js
    └── icon_cache/ (画像キャッシュフォルダ)
```

## 🔄 データ更新と自動運用の仕組み

Steamコミュニティマーケットの相場変動に対応するため、自動同期の仕組みを導入しています。

### A. ローカルまたはVPS等で手動更新する場合
1. `node` (Node.js環境) をインストールします。
2. コマンドプロンプトまたはターミナルを開きます。
3. 必要に応じて `scripts/daily_sync.js` にSteamセッションのCookieを設定します（取引履歴を大量に取得する際にIP制限を避けるため）。
4. 以下のコマンドを実行してデータを更新します。
   ```bash
   node scripts/daily_sync.js
   ```
5. 更新された `data/market_db.js` をWebサーバーに再アップロードします。

### B. GitHub Actionsを使用した完全自動同期（推奨）
本リポジトリにはすでに自動同期用ワークフロー（`.github/workflows/daily-sync.yml`）が組み込まれています。
GitHubリポジトリにプッシュし、以下の設定を行うだけで毎日自動で価格データが更新されます。

#### 設定手順:
1. プロジェクトをGitHubのプライベートまたはパブリックリポジトリにプッシュします。
2. リポジトリの **Settings** > **Secrets and variables** > **Actions** に移動します。
3. **New repository secret** をクリックし、以下を登録します。
   - **Name**: `STEAM_COOKIE`
   - **Value**: あなたのSteamコミュニティマーケットにログインした際のブラウザのCookie（セッションIDなどを含む文字列）
4. これにより、毎日自動で最新価格が取得され、リポジトリ内の `data/market_db.js` が自動更新されてコミット＆プッシュされます。GitHub Pages等でホスティングしている場合は自動的にサイトにも最新情報が反映されます。

## ⚠️ 公開にあたっての注意事項

- **Cookieの管理**: `scripts/` 内のスクリプトにログイン情報（Cookie）を直接ハードコードしてパブリックなGitHubにプッシュしないように注意してください。本リポジトリは環境変数から自動読み込みするセキュアな設計に改良されています。
- **商標・ファンアートポリシー**: 本ツールは「タスクバーヒーロー（Taskbar Hero）」のファンメイドツールです。画像データ等の著作権はゲーム開発元に帰属します。
